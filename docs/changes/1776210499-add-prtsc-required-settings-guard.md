# PrtSc 필수 설정 누락 가드 추가

## 목적

- PrtSc를 눌렀을 때 번역 실행에 필요한 필수 설정이 비어 있으면
  캡처/오버레이/OCR을 시작하지 않고 즉시 사용자에게 안내한다.

## 구현 계획

1. PrtSc 처리 시작 전에 현재 설정 스냅샷에서 필수 값 누락 여부를 검사한다.
2. 누락 시 설정 창을 표시하고 어떤 항목을 채워야 하는지 하이라이트와 함께 안내한다.
3. 설정이 누락된 경우 캡처 진행을 중단하고 busy 플래그를 즉시 복구한다.
4. 설정 저장 시에도 필수 항목 누락이면 저장을 거부한다.
5. 필수 설정 판별 로직은 테스트로 고정한다.

## 설계

### 필수 설정 정의

- 현재 번역 호출에 실질적으로 필요한 값은 아래 두 항목이다.
  - `AI_GATEWAY_API_KEY`
  - `AI_GATEWAY_MODEL`
- `SYSTEM_PROMPT`는 기본값이 있어 누락으로 보지 않는다.
- `OCR_SERVER_DEVICE`는 기본값 `cpu`가 있으므로 누락으로 보지 않는다.

### 사용자 안내 방식

- PrtSc 입력 시 필수 설정이 누락되면 설정 창을 앞으로 가져오고 포커스를 준다.
- 설정 창 상단 상태 영역에 안내 메시지를 표시한다.
- 누락된 설정 항목 제목과 입력 컨트롤을 강조 표시해 사용자가 바로 수정할 수 있게 한다.
- 이 경우 오버레이는 띄우지 않으며 스크린샷 캡처도 수행하지 않는다.

### 저장 검증

- 설정 저장 직전에도 동일한 필수 항목 검사를 적용한다.
- 필수 항목이 비어 있으면 `.env` 저장과 메모리 설정 반영을 모두 중단한다.
- 저장 실패 시 설정 창은 닫지 않고, 누락된 항목 하이라이트를 유지한다.
